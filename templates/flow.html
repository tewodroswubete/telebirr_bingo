System Components

Component Technology Purpose
Web Server Python Flask (port 1111) Serves the form and handles validation requests
Text Parser Python Regex Extracts transaction number, amount, sender from SMS text
Receipt Fetcher Python Requests Downloads receipt PDF from transactioninfo.ethiotelecom.et
PDF OCR PyMuPDF + Tesseract Converts image-based receipt PDF to readable text
Receipt Parser Python Regex Extracts Payer, Credited Party, Amount, Status from receipt
Validator Python Checks Credited Party = your account, Status = Completed
Duplicate DB SQLite (transactions.db) Stores validated transactions to prevent reuse
Frontend HTML / CSS / JavaScript Form input + popup results (success / error / duplicate)

Validation Levels

Level When What it checks
Level 1: PDF Receipt Receipt PDF fetched + OCR works Credited Party name + phone matches your account, Status = Completed, Amount extracted
Level 2: SMS Fallback Receipt PDF unavailable (network/VPN) Message type = "received", Account holder = your name, Amount extracted
Duplicate Check Always (before any validation) Transaction number not already in database
